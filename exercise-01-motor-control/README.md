# Exercise 01 – Motor Control with Sensors and Button

## Description
Practical exercise implementing motor control using sensors and button with AVR microcontroller.

## Demo
[![Watch the demo](https://img.youtube.com/vi/7NhNgfxXN-0/0.jpg)](https://www.youtube.com/watch?v=7NhNgfxXN-0)

## Components
- AVR ATmega
- Motor
- Sensors
- Button

## How to compile
```bash
avr-gcc -mmcu=atmega328p -o main.elf main.c
```
